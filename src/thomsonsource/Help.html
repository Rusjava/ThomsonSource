TSource application user guide

The TSource application is intended as tool to make a fast estimate of the main Thompson source parameters such as: the X-ray photon flux in the given direction, spectral brilliance in a given direction.

The application consists of the main window, where one can specify the parameters of the Thompson source as well as plot 2D and 1D flux, energy and spectral brilliance distributions in the form of color plots, and two calculation boxes where one can obtain and plot estimates of the geometric factor and brilliance in the forward direction as function of a number of parameters.

The parameters from the main window can be saved to the file and loaded from the file. The plots from the calculation boxes can be save to a file as well.

Parameters

Electron bunch parameters

  • Electron energy – the average energy of the electrons in MeV
  • Charge – the electric charge of the bunch in nQ
  • Spread – the relative energy spread of the electrons
  • Length – the length of the electron bunch in ps
  • Emittance – the emittance of the bunch in mm×mrad
  • Beta x – the beta function at the point of interaction in the x direction
  • Beta y – the beta function at the point of interaction in the y direction

Laser pulse parameters

  • Photon energy – the average energy of the photons in eV
  • Pulse energy – the energy of the laser pulse in mJ
  • Length – the length of the laser pulse in ps
  • Rayleigh length – the Rayleigh length of the laser pulse in mm
  • Pulse frequency – the number of laser pulses per second in MHz
  • Delay – the temporal delay of the laser pulse relative to the electron bunch in ps

Relative position

  • X-shift – the spatial shift of the electron bunch relative to the laser pulse in x direction in mm
  • Y-shift – the spatial shift of the electron bunch relative to the laser pulse in y direction in mm
  • Z-shift – the spatial shift of the electron bunch relative to the laser pulse in z direction in mm
  • Angle – the angle between direction of the electron bunch and the laser pulse in mrad. Rotation is thought to occur about x-axis

Graphs

X-ray flux tab

The left color plot shows the X-ray photon flux (in mm−2×s−1) in a particular direction relative to the direction of the electron bunch. The right color plot shows the X-ray photon spectral flux (in mm−2×s−1×0.1%) for a particular angle and particular energy at the angular position corresponding to the slider position below the left plot.

X-ray energy tab

The left color plot shows the X-ray photon energy (in keV) in a particular direction relative to the direction of the electron bunch. The right plot shows the cross-section X-ray photon energy (in keV) at the angular position corresponding to the slider position below the left plot.

File menu

This menu allows one to save (File -> Save parameters) the parameters from the main window to a file and load (File -> Load parameters) them from a file.

Calculation menu

The Calculation menu provides access to two calculations boxes: for the spectral brilliance in the direction of the electron bunch and the geometric factor. These quantities are calculated by pressing Calculate button and can be plotted as a function of a number of parameters, which can be chosen from a drop-down box. The desirable parameter range can be entered via two text fields. All other necessary parameters are taken from the main window of the program. The results of calculation can be saved as a text file using Save button.

The brilliance calculation box also allows to choose whether the transversal velocity spread of the electrons is taken into account.

Shadow menu

The Shadow menu allows one to generate a set of rays for the use as an X-ray source in Shadow ray-tracing software. To choose the number of generated rays use Shadow -> Parameters dialog

.

Option menu

Option menu provides access to additional graph parameters via Options -> Graph parameters as well as to the Monte-Carlo integration parameters via Options -> Monte-Carlo parameters. Options -> Velocity spread specifies whether the traversal velocity spread is taken into account in the main window calculations and also while generating a set of rays for Shadow.
